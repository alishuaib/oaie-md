```py
sorted_numbers = sorted(numbers)
print("Sorted array (ascending):", sorted_numbers)  # Output: Sorted array (ascending): [1, 2, 5, 8, 9]

# Descending order
sorted_descending = sorted(numbers, reverse=True)
print("Sorted array (descending):", sorted_descending)  # Output: Sorted array (descending): [9, 8, 5, 2, 1]
```
